So why a blog?

Mainly because I've wanted a space for a while that I can put down some thoughts and occasionally cross share. Writing up about a topic is a great way to learn it.

What is this built with?

I wanted this to be an opportunity to use some technology that I've heard about and see what they were like for the most part. For those who are interested; it includes the following:

  1. Qwik, specifically resumability caught my attention and I'm intrigued to see how Qwik develops. A 1.0 RC was announced yesterday but time will tell as there's a lot going on in the community (e.g. Remix, RSC, Astro, Deno, Bun and the many others that I've not listed here) at the moment. While the innovation is promising; trying to keep up to date with everything can fatiguing.
  2. Tailwind, I hadn't intended to learn this, but I needed a CSS framework. I have seen people comment above how much they like Tailwind and now see why.
  3. Playwright, I'd normally default to cypress for E2E testing in JS / TS but again had heard good things and given the recent announcement of page.getBy\* I thought I'd give a whirl. Getting axe-core integrated for some automated accessibility testing was straight forward as well.
  4. Github pages, is a convenient way to host for free.

It's been nice to be able to leverage Github actions to automate the build and deploy process.

We'll see what else pops up on this space.
